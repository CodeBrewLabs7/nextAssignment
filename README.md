## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```


## Check Test Cases 
```bash
npm run test login 
#and 
npm run test dashboard
#and 
npm run test projects

```


Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
